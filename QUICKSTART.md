# ⚡ Быстрый старт

Запустите бота за 5 минут!

## 1️⃣ Получите токен бота

1. Откройте [@BotFather](https://t.me/botfather) в Telegram
2. Напишите `/newbot`
3. Следуйте инструкциям
4. Скопируйте токен

## 2️⃣ Установка

```bash
# Windows
git clone https://github.com/your_repo/legitVPN.git
cd legitVPN
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt

# Linux/Mac
git clone https://github.com/your_repo/legitVPN.git
cd legitVPN
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## 3️⃣ Конфигурация

```bash
# Скопируйте пример конфига
cp .env.example .env

# Отредактируйте .env файл
# - BOT_TOKEN: вставьте токен от BotFather
# - ADMIN_IDS: ваш Telegram ID
# - MARZNESHIN_API_URL: адрес вашей панели
```

## 4️⃣ Запуск

```bash
python main.py
```
